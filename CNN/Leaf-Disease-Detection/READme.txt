Required Python modules 
1.Flask
2.Tensorflow
3.Numpy
4.Matplotlib
5.Pandas
6.Pillow
7.os
Before running ensure all of the above Modules are installed in the virtual environment
The below mentioned should be in one directory
1)leaf.py
2)leaf.ipynb
3)templates (folder) // this will include the background.jpg,index.html and result.html
4)Trained_leaf_Detection.keras

The data was used from Kaggle: https://www.kaggle.com/datasets/vipoooool/new-plant-diseases-dataset
